# subname
一款基于subDomainsBrute子域名爆破工具结果的处理脚本
## 使用帮助
### 功能介绍：
subname可将lijiejie撰写的工具subDomainsBrute的结果进行自动化处理。统计子域名所处的C段信息，以及子域名的可否被用户访问。
### 依赖环境：
    pip install requests
### 使用方法：
    python subname.py citi.log
